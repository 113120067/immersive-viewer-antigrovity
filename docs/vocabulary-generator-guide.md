# 英文單字學習圖片生成器使用指南

## 🎯 新功能概覽

### 1. **API 提供商選擇**
現在你可以選擇使用哪個 AI 來生成圖片：
- **🔷 Azure OpenAI DALL-E**: Microsoft 的 AI 圖片生成服務
- **🟢 Google Imagen**: Google 的 AI 圖片生成服務  
- **🤖 Auto**: 系統自動選擇最佳可用的服務

### 2. **預設設置（按你的要求）**
系統預設為最適合學習的設置：
- **🎨 風格**: 卡通風格 (Cartoon) - 友善有趣
- **🎯 級別**: 初學者 (Beginner) - 簡單清晰
- **🧠 記憶技巧**: 故事記憶 (Story-based) - 容易記住

### 3. **快速設置按鈕**
- **🎯 Reset to Default**: 恢復到你要求的預設設置
- **👶 Kid Friendly**: 兒童友善設置（卡通 + 情感記憶）
- **🎓 Advanced Mode**: 進階設置（真實情境 + 聯想記憶）

## 📋 使用步驟

### 基本使用
1. **登入系統** - 訪問 `http://localhost:3000/vocabulary-generator`
2. **輸入單字** - 在 "English Word" 欄位輸入要學習的單字
3. **選擇 API** - 選擇想使用的 AI 提供商（或保持 Auto）
4. **點擊生成** - 系統會自動使用最佳設置生成學習圖片

### 進階設置
1. **調整風格** - 根據需要選擇不同的視覺風格
2. **設定級別** - 根據學習者程度調整複雜度
3. **選擇記憶技巧** - 使用不同的記憶策略
4. **快速設置** - 使用預設按鈕快速應用常用設置

## 🔍 功能詳解

### API 提供商比較
| 提供商 | 狀態 | 特色 | 適用場景 |
|--------|------|------|----------|
| Azure OpenAI | ✅ 可用 | 高品質、多樣化風格 | 需要精細控制和高品質圖片 |
| Google Imagen | ⚠️ 開發中 | 自然真實、快速生成 | 暫時不可用，請使用 Azure |
| Auto | ✅ 可用 | 智能選擇、備援機制 | 一般使用，目前會自動選擇 Azure |

**注意**: Google Imagen 功能正在開發中，因為 Google Gemini API 目前不支援直接的圖片生成功能。建議使用 Azure OpenAI DALL-E。

### 學習風格說明
- **Memory Aid**: 清晰的教育插圖，重點突出
- **Cartoon**: 友善的卡通風格，適合兒童和初學者
- **Real Context**: 真實情境，幫助理解使用場合
- **Story Scene**: 故事場景，增強記憶效果
- **Concept Diagram**: 概念圖表，適合抽象概念

### 記憶技巧解釋
- **Visual Impact**: 視覺衝擊，使用鮮明色彩和對比
- **Association**: 聯想記憶，建立單字與其他概念的連結
- **Emotional**: 情感記憶，喚起正面情感增強記憶
- **Story-based**: 故事記憶，通過敘事情節幫助記憶
- **Contrast**: 對比記憶，使用對比突出單字含義

## 📊 系統監控

### 查看 API 狀態
- **狀態頁面**: `http://localhost:3000/image-generator/status`
- **即時監控**: 頁面頂部顯示可用的 AI 提供商
- **歷史記錄**: 查看之前使用的提供商和設置

### 提示詞透明度
- **完整提示詞**: 可查看系統生成的完整 AI 提示詞
- **提示詞分析**: 了解系統如何構建學習導向的提示詞
- **組件標籤**: 彩色標籤顯示提示詞的不同組成部分

## 💡 使用建議

### 最佳實踐
1. **從預設開始**: 使用預設的卡通風格、初學者級別、故事記憶
2. **嘗試不同 API**: 比較 Azure 和 Google 的生成效果
3. **調整級別**: 根據學習者程度逐步提高複雜度
4. **查看提示詞**: 了解系統如何構建教育性提示詞

### 故障排除
- **API 不可用**: 系統會自動切換到備用提供商
- **生成失敗**: 檢查網路連接和認證狀態
- **品質不佳**: 嘗試不同的提供商或調整設置
