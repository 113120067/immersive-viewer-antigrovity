# 📱 Kids Vocabulary 響應式設計指南

## 📋 設計概述

Kids Vocabulary 現在採用完全響應式設計，能夠在手機、平板和桌面設備上提供最佳的用戶體驗。

## 🎯 響應式斷點

### 📱 手機版 (< 576px)
- **布局**：單欄垂直堆疊
- **特色**：大型觸控按鈕、簡化文字、緊湊間距
- **優化**：觸控友善、單手操作

### 📟 平板版 (576px - 992px)
- **布局**：兩欄混合布局
- **特色**：中等按鈕大小、適中文字、平衡間距
- **優化**：橫向和縱向支援

### 🖥️ 桌面版 (≥ 992px)
- **布局**：多欄水平排列
- **特色**：標準按鈕、完整文字、寬鬆間距
- **優化**：滑鼠懸停效果、鍵盤導航

## 🔧 技術實現

### Bootstrap 響應式網格
```pug
.container-fluid.px-2.px-md-3.mt-2.mt-md-4
  .row.g-2.g-md-3.justify-content-center
    .col-12.col-lg-8.order-2.order-lg-1
      // 主要內容區域
    .col-12.col-lg-4.order-1.order-lg-2
      // 側邊欄區域
```

### 響應式類別使用
- **間距**：`p-2.p-md-4` (手機 8px，桌面 24px)
- **字體**：`fs-5.fs-md-4` (手機較小，桌面較大)
- **顯示**：`d-none.d-sm-block` (手機隱藏，平板顯示)
- **排序**：`order-1.order-lg-2` (手機優先，桌面重排)

### CSS 媒體查詢
```css
/* 手機版樣式 */
@media (max-width: 575.98px) {
  .container-fluid {
    padding-left: 8px;
    padding-right: 8px;
  }
  
  #wordInput {
    font-size: 14px;
    border-radius: 15px;
  }
  
  #generateBtn {
    min-width: 60px;
    font-size: 14px;
  }
}

/* 平板版樣式 */
@media (min-width: 576px) and (max-width: 991.98px) {
  #wordInput {
    font-size: 17px;
  }
}

/* 桌面版樣式 */
@media (min-width: 992px) {
  #wordInput {
    font-size: 18px;
    border-radius: 25px;
    border-width: 3px;
  }
}
```
